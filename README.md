# Astro Blog

A static blog built with Astro, Decap CMS, and deployed on Cloudflare Pages.

## Local Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Preview

```bash
npm run preview
```

## Content Editing via Decap CMS

1. Site must be deployed and accessible over HTTPS
2. Visit `/admin`
3. Login with GitHub
4. Create and edit posts

## Cloudflare Pages Deployment

- **Build command:** `npm run build`
- **Output directory:** `dist`
